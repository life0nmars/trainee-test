# Решение задачи 2
Структура простая - папка срезультатами `results/` и входными данными `vina_data/`. Решение представлено в виде ноутбука `solution.ipynb`\
При написании кода не было цели обработать все исключения, как и написано в задании. Рассматривалось то, что было прямо обозначено (наличие водородов, молекулы в 2d) :-) \
Для реального использования, как минимум, надо было бы сразу проверять содержится ли в .sdf полная информация о молекуле (3d + Hydrogenes). Однако тут я посчитала это избыточным. Ничего не ломается, если на вход поступает "полный" .sdf: координаты будут перегенерированы, что все еще допустимо для последующего докинга.
